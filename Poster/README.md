# Project Poster

## Title
Computational Analysis of unknown sequence by Using Bioinformatics Tools

## Description
This folder contains the scientific poster summarizing the complete bioinformatics analysis of ORF 6, including DNA, RNA, Protein, Phylogenetic, and Protein–Protein Interaction analyses.

## Poster Contents
The poster includes the following sections:

1. Introduction  
2. DNA Analysis (ORF identification, GC content, BLAST results)  
3. RNA Analysis (UTRs, secondary structure, minimum free energy)  
4. Protein Analysis (physicochemical properties, domains, structure)  
5. Subcellular Localization  
6. Phylogenetic Analysis  
7. Protein–Protein Interaction Analysis  
8. Tools
